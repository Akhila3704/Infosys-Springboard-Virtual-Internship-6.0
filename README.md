# Infosys Springboard Virtual Internship 6.0  
## PulseScore – Healthcare Operations Dashboard

### Project Overview
Springboard Virtual Internship 6.0.
It provides a unified view of healthcare infrastructure and operational readiness across India, empowering stakeholders with actionable insights for better decision-making.



### Objectives
- Assess healthcare infrastructure availability at the state and district levels

- Highlight resource gaps and operational bottlenecks

- Deliver data-driven insights through dynamic, interactive visualizations

### Dataset
- Curated healthcare dataset covering hospitals, beds, and infrastructure indicators
- GeoJSON files for state and district-level mapping

### Technology Stack
- Programming Language: Python
- Framework: Streamlit
- Libraries: Pandas, NumPy, Plotly, Matplotlib
- Data Formats: CSV, GeoJSON 

### Project Structure
- `app.py` – Core Streamlit application 
- `dataset/` – Cleaned and processed healthcare data
- `geojson/` – State and district boundary files 
- `logic/` –  Data processing scripts and utility functions  
- `sections/` –  Modular dashboard components
- `requirements.txt` – Python dependencies 

### How to Run the Project
1. Install dependencies: pip install -r requirements.txt
2. Run the application: streamlit run app.py

### Internship Context
This project demonstrates the practical application of data cleaning, analytics, and dashboard development in the healthcare domain.
It reflects the integration of technical skills with real-world problem-solving, showcasing how interactive dashboards can support operational intelligence and executive decision-making.
